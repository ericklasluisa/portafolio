<h2>🛠️ Pasos de instalación:</h2>

<p>1. Clonar el repositorio</p>

```
git clone https://github.com/ericklasluisa/Gestion-Vehicular-React.git
```

<p>2. Instalar las dependencias</p>

```
npm install
```

<p>3. Ejecutar la aplicación</p>

```
npm start
```
